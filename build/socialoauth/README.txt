A Package For SNS Sites With OAuth2 Support.

Author: Wang Chao

Author_email: yueyoum@gmail.com


Supported the following sites (and the list are growing):

*   renren      (人人)
*   tencent     (腾讯)
*   douban      (豆瓣)
*   sina        (新浪微博)
*   netease     (网易微博)
*   sohu        (搜狐微博)
*   baidu       (百度)
*   kaixin001   (开心网)
*   taobao      (淘宝网)
